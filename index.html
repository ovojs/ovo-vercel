OvO Vercel

Backend for OvO using Vercel functions (experimental).

    
GET https://ovo-vercel.vercel.app/api/comment?domain=a.cn&path=/posts/1&page=0 HTTP/1.1
    
  
    
POST https://ovo-vercel.vercel.app/api/comment HTTP/1.1
Content-Type: application/json

{
  "domain": "a.cn",
  "path": "/posts/1",
  "content": "hahaha",
  "issuer": "dev",
  "issuer_website": "a.cn",
  "issuer_email": "i@a.cn"
}
